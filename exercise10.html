CSS Selectors

CSS selectors are used to "find" (or select) the HTML elements you want to style.

Types of Selectors

  • Universal Selector (*)
  • Type Selector (element)
  • Class Selector (.class)
  • ID Selector (#id)
  • Attribute Selector ([attribute])
